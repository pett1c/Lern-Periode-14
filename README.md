# Eventify – Fullstack Event Management Platform

## Beschreibung

**Eventify** ist eine moderne Event-Management-Plattform, entwickelt mit dem **MERN Stack** (MongoDB, Express, React, Node.js) und **Socket.io** für Realtime-Chat.

Die Anwendung ermöglicht:

- Veranstaltern, Events zu erstellen, Tickets zu verkaufen und Teilnehmerstatistiken einzusehen  
- Nutzern, Tickets zu buchen, Events zu durchsuchen und direkt im Event-Chat mit anderen Teilnehmern zu kommunizieren  
- Admins, die Plattform zu überwachen und globale Statistiken zu verwalten  

Das Projekt kombiniert **CRUD-Operationen**, **Realtime-Kommunikation**, **Rollen & Berechtigungen** sowie **interaktive Dashboards**.

---

## Features

**Backend**

- JWT Auth + Rollenbasierte Zugriffskontrolle (Admin, Organizer, User)  
- REST API für Events, Tickets und Chat  
- MongoDB mit Mongoose für persistente Speicherung  
- Socket.io für Event-spezifischen Gruppenchat  
- QR-Code für Tickets (optional erweiterbar)  

**Frontend**

- React + React Router  
- Axios für API-Integration  
- Chatfenster mit Realtime-Nachrichten  
- Dashboard mit Eventübersicht, Ticketstatus und Analytics  
- Responsive Design für Desktop & Mobile
